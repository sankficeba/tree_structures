# README: Визуализатор деревьев поиска

## Описание
Этот проект реализует **визуализацию** четырех типов сбалансированных деревьев поиска:
- **AVL-дерево** (AVL Tree)
- **Красно-черное дерево** (Red-Black Tree)
- **Декартово дерево (Treap)** (CT Tree)
- **Splay-дерево** (Splay Tree)

Программа позволяет **добавлять**, **удалять** и **генерировать** случайные вершины, а также **управлять масштабированием и перемещением сцены**.

---

## Функционал
- **Добавление узла** в дерево вручную
- **Удаление узла** по клику
- **Генерация случайных узлов**
- **Визуализация структуры дерева**
- **Выбор структуры данных для отображения**
- **Масштабирование и перемещение сцены**

---

## Технологии
Проект написан на **C++** с использованием библиотеки **SFML** для графической визуализации.

---

## Установка и запуск
### Требования:
- **C++ компилятор** (g++, MSVC и т.д.)
- **SFML 2.5+** (https://www.sfml-dev.org/download.php)


### Скачать готовый `.exe`
Если вы не хотите собирать проект вручную, вы можете скачать **готовый исполняемый файл** вместе с библиотеками из раздела **Releases** на GitHub.

---

## Управление
- **Выбор структуры дерева:**
  - Нажмите **"AVL"**, **"RBT"**, **"CT"**, **"Splay"** внизу экрана.
- **Добавление узла вручную:**
  - Нажмите **"Enter the vertex key"** и введите число.
- **Добавление случайных узлов:**
  - Нажмите **"Enter the number of generated vertices"** и введите количество.
- **Удаление узла:**
  - Кликните на вершину в дереве.
- **Масштабирование сцены:**
  - Прокрутите колесико мыши вверх/вниз.
- **Перемещение сцены:**
  - Зажмите **левую кнопку мыши** и двигайте сцену.
